# 6.用canvans画空心三角形 #

    `context.beginPath();//开始路径
        context.moveTo(100,100); //三角形，左顶点
        context.lineTo(300,100);//右顶点
        context.lineTo(300,500);//底部的点
        context.closePath();  //结束路径
        context.stroke();  //描边路径 
`
