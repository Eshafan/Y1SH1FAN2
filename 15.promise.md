# 一、异步编程的几种模式 #
## 一、回调函数形式（缺点：引起回调地狱） ##
    `function f1(callback){
	    callback();
	}
	function f2(callback){
	    callback();
	}
	function f3(callback){
	    callback();
	}
	f1(f2(f3))
	关键就是一步一步的嵌套调用，callback的意思就是等它完成后再执行`
## 二、发布订阅者模式 ##
    `let dep = {
	  list: [],
	  on: function (fn) {
	    list.push(fn);
	  },
	  emit: function () {
	    this.list.forEach(event => {
	      typeof event === 'function' ? event() : null;
	    })
	  }
	};`
